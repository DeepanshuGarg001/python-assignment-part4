# Student Performance Analysis & Prediction

This project involves analyzing a student performance dataset, producing visualizations, and building a machine learning model to predict student success.

## Project Structure
- `part4_visualization_ml.ipynb`: Main Jupyter notebook with analysis and modeling.
- `students.csv`: Dataset containing student records.
- `plot*.png`: Generated visualizations.

## Requirements
- pandas
- matplotlib
- seaborn
- scikit-learn
